# Jesus Devotional Channel Guide

## Overview
Compilation videos with 5-10 worship songs, each paired with looping visuals.
**Duration:** 25-60 minutes total (3-6 min per song)

---

## Production Workflow

### 1. Audio - Song Generation
**Tool:** Suno AI

**Styles:** Contemporary worship, acoustic, traditional hymns, instrumental, gospel, Gregorian chant

**Prompt Examples:**
- "Acoustic Christian worship, gentle guitar, heartfelt vocals about Jesus' love"
- "Contemporary praise song, uplifting melody about grace and redemption"
- "Soft worship ballad, piano and strings, intimate lyrics about faith"
- "Gospel worship, soulful vocals, powerful salvation message"

**Thematic Collections:** Morning devotional, evening prayer, grace/mercy, praise/thanksgiving, comfort, salvation

### 2. Visuals - Looping Videos
**Image Tools:** Midjourney, DALL-E, Leonardo AI, Whisk

**Themes:** Jesus imagery, cross/symbols, Biblical landscapes, nature scenes, churches, open Bible, stained glass, candlelit prayer, heavenly/ethereal

**Style:** Reverent imagery, warm colors (gold, amber, soft blues), universal Christian symbolism

**Video Tools:** Runway Gen-3 Flow, Pika Labs
- Use same image for first/last frame (perfect loop)
- Subtle motion: slow zoom, gentle pan, light rays, flickering candles
- 5-10 second loop extended throughout song

### 3. Assembly & Editing
1. Arrange songs in meaningful order
2. Match each song with thematic visual
3. Add 2-3 second cross-dissolves between songs
4. Create YouTube timestamps
5. Add text overlays (song titles, optional Bible verses)
   - Fonts: Cinzel, Cormorant, Playfair Display
   - White/gold text, 5-8 seconds at song start
